# Loops Examples

Run this file from the project root:

```bash
go run loops/loops.go
```

## What To Practice

- Use `for init; condition; post` for counted loops.
- Use `for condition` when you want while-style behavior.
- Use `break` to leave a loop early.
- Use `continue` to skip the current iteration.

## Useful Predefined Functions

```go
len(items) // Get collection length so the loop knows where to stop.
fmt.Printf("Index: %d\n", i) // Print loop state while learning or debugging.
time.Since(start) // Measure how long a loop or example took.
```

Use this folder when you want to repeat work safely and control when repetition stops.
