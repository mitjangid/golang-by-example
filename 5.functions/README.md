# Functions Examples

Run this file from the project root:

```bash
go run functions/functions.go
```

## What To Practice

- Create functions with parameters and return values.
- Return multiple values from one function.
- Pass a function as an argument.
- Use variadic parameters with `...`.
- Create closures that remember values from an outer scope.

## Useful Predefined Functions

```go
fmt.Println(value) // Print function input or output while learning.
fmt.Printf("%T\n", value) // Print the type of a value passed to a function.
panic("stop") // Stop immediately for unrecoverable demo errors; avoid in normal flow.
recover() // Catch a panic inside a deferred function when you intentionally demonstrate recovery.
```

Use this folder when you want to organize repeated logic into small reusable actions.
